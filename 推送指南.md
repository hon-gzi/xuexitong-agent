## 📋 推送到 GitHub 的两种方法

### 方法一：通过 GitHub 网页（推荐新手）

#### 步骤 1: 在 GitHub 创建新仓库

1. 打开浏览器，访问 https://github.com/new
2. 填写仓库信息：
   - **Repository name**: `xuexitong-agent`（或你喜欢的名字）
   - **Description**: `学习通智能助手 - 基于 LLM 的自动化工具`
   - **Visibility**: 选择 Public（公开）或 Private（私有）
   - **⚠️ 重要**: ❌ **不要勾选** "Add a README file"
   - **⚠️ 重要**: ❌ **不要勾选** "Add .gitignore"
   - **⚠️ 重要**: ❌ **不要勾选** "Choose a license"
3. 点击 **Create repository**

#### 步骤 2: 推送代码到 GitHub

创建仓库后，GitHub 会显示命令。在你的项目目录执行：

```bash
# 添加远程仓库（替换 YOUR_USERNAME 为你的 GitHub 用户名）
git remote add origin https://github.com/YOUR_USERNAME/xuexitong-agent.git

# 推送代码
git branch -M main
git push -u origin main
```

---

### 方法二：使用 GitHub CLI（推荐熟练用户）

#### 步骤 1: 安装 GitHub CLI

**Windows:**
```bash
# 使用 winget
winget install GitHub.cli

# 或使用 Chocolatey
choco install gh
```

**macOS:**
```bash
brew install gh
```

**Linux:**
```bash
# Debian/Ubuntu
sudo apt install gh

# Fedora
sudo dnf install gh
```

#### 步骤 2: 登录 GitHub

```bash
gh auth login
```

按提示操作：
1. 选择 `GitHub.com`
2. 选择 `HTTPS`
3. 选择 `Login with a web browser`
4. 复制显示的 one-time code
5. 按 Enter 打开浏览器，粘贴 code 完成登录

#### 步骤 3: 创建仓库并推送

```bash
# 创建仓库（PUBLIC 公开 / PRIVATE 私有）
gh repo create xuexitong-agent --public --source=. --push

# 或创建私有仓库
gh repo create xuexitong-agent --private --source=. --push
```
